## Installing the VPN Client

### Windows
1. Download the VPN client from the IT portal
2. Run the installer as administrator
3. Follow the installation wizard
4. Restart your computer when prompted

### Mac
1. Download the VPN client from the IT portal
2. Open the .dmg file
3. Drag the application to your Applications folder
4. Launch the application

## Connecting to VPN

1. Open the VPN client
2. Enter your username and password
3. Select the **"Corporate Network"** server
4. Click **Connect**
5. Wait for the green "Connected" status

## Troubleshooting

**Can't connect?**
- Check your internet connection
- Verify your credentials
- Try a different server location
- Contact IT if issues persist

**Slow connection?**
- Try connecting to a different server
- Close unnecessary applications
- Check for background downloads
